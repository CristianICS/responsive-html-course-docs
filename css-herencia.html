Clase

El método cuyos estilos se priorizan por encima de los asociados al elemento root y a las etiquetas HTML es aplicar un estilo definiendo una clase. Al crear una clase que contenga la propiedad color: pink y aplicarla a un elemento h2, se sustituye el color de este último por el rosa. Se deja de tener en cuenta el color aplicado anteriormente.

Si existen varias clases con la misma propiedad aplicadas a un objeto, la preferencia la tiene aquella que ha sido definida en último lugar.
