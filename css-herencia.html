Herencia

Un elemento puede tener asignados a la vez varios estilos. Estos pueden contener propiedades idénticas que entren en conflicto. Establecer cuál de ellas tiene prioridad se consigue mediante las reglas de herencia de CSS, marcando el estilo a mostrar en función del método con el que fue aplicado. De menor a mayor prioridad los métodos son:
