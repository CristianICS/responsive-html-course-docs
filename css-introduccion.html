Conceptos básicos

Las hojas de estilo en cascada o Cascading Style Sheets pasan al navegador la forma en la que representar los elementos de la web, escritos en HTML.

Advertencia

El lenguaje CSS dintingue entre mayúsculas y minúsculas, i.e., case-sensitive. Tener cuidado con el uso de mayúsculas.

CSS ha sido adoptado por la mayoría de navegadores, permitiendo al usuario controlar:

  • color
  • fuente
  • posicionamiento
  • espaciado y sangría
  • tamaño
  • decorazión
  • transiciones y animaciones

La idea detrás de CSS es poder utilizar un selector para localizar un elemento HTML en el DOM y aplicarle después una variedad de atributos que modifiquen su representación en la página.
