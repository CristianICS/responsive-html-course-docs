Mejorar accesibilidad

Atajos de teclado

Los elementos HTML pueden integrar el atributo accesskey, donde puede añadirse un comando específico que, al pulsarlo, active el elemento y permita seleccionarlo. Puede utilizarse en cualquier elemento, pero es particularmente útil cuando se aplica con los interactivos, e.g., links, formularios etc.

Este tipo de atajos permiten una navegación más eficiente a los usuarios que solo usen el teclado.

Tabulación

El atributo tabindex permite establecer el orden por el que se seleccionarán los elementos al pulsar la tecla tab.

Por defecto, solo algunos elementos (e.g. links y formularios) tienen activada esta opción, y al tabular la página se centra en ellos. El orden se corresponde con el orden de aparición en la web.

Al establecer dentro de un elemento el atributo tabindex="0", permitirá que este aparezca en el flujo de tabulación. El orden de selección se modifica añadiendo valores enteros > 1

Advertencia

Un valor tabindex negativo (normalmente -1) indica que un elemento es resaltable al tabular, pero no es alcanzable por el teclado. Este método suele utilizarse para atraer la atención sobre un contenido, e.g., cuando un elemento <div>, utilizado como ventana emergente, es activado.
