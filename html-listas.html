Listas

Permiten agrupar un grupo de elementos de diferentes formas.

Listas sin ordenar

Comienzan con la etiqueta <ul>. Cada item empezará con la etiqueta <li>.

Los elementos irán precedidos por un pequeño círculo.

Listas ordenadas

Comienzan con la etiqueta <ol>. Cada item empezará con la etiqueta <li>.

Los elementos irán precedidos por números comenzando por el uno. No es posible continuar automáticamente la numeración en varias listas ordenadas separadas ni ocultar la numeración de algunos items. Si es posible modificar el número de un elemento modificando su atributo value. La numeración continúa a partir de este nuevo valor.

Listas de definiciones

Una lista de definiciones se compone de términos y sus acepciones

Comienzan con la etiqueta <dl>. Los términos o nombres se engloban dentro de la etiqueta <dt> y sus definiciones dentro de <dd>

Listas anidadas

Pueden encadenarse listas incluyendo una nueva etiqueta ul u ol dentro de un elemento <li> anterior, cerrándo esta nueva lista justo antes del cierre de </li>.
