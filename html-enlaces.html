Enlaces

Se escriben entre etiquetas <a></a>. Sus atributos son:

  • href Integra la dirección a la que remite el enlace
  • target Especifica donde debe abrirse la dirección del enlace. Por defecto se muestra en la misma pestaña del navegador. Este comportamiento puede modificarse con el atributo target, añadiendo uno de los siguientes valores:
    • "_self" Valor por defecto. Abre el documento en la misma ventana en la que es activada.
    • "_blank" Abre el enlace en una pesataña nueva.
    • "_parent" Abre el documento en el parent frame.
    • "_top" Abre el enlace en la totalidad de la ventana.
Accesibilidad

El texto al que engloban las etiquetas del enlace debe describir o indicar el contenido del mismo, i.e.,

  • <a href="wikipedia.com" target="_blank">Wikipedia entry</a>

Los usuarios con lectores de pantalla tienen diferentes opciones para mostrar el contenido de la web, entre las que se incluye la opción de escuchar los enlaces de la página. Se lee el texto dentro de las etiquetas, siendo de poca utilidad listas llenas de enlaces "click here" o "read more".

Tipos de enlaces

Los enlaces pueden hacer referencia tanto a contenido externo (otras webs o documentos) como a contenido interno (dentro del mismo documento HTML).

  • Enlaces internos el atributo href comienza con una almohadilla # seguida del id que describe el elemento a enlazar.

<div id="contacts">...</div>

<a href="#contacts">Contacts</a>

  • Enlaces externos
    • Dirección web: <a href="https://freecodecamp.org">fcc home page</a>
    • Dirección de archivo: <a href="/section/main.html">good article</a>
    • Dirección de correo: <a href="mailto:someone@example.com">Send email</a>
  • Enlace nulo href="#"

Convertir un elemento HTML en enlace

Los enlaces pueden integrar otros elementos, e.g. imágenes, para que al pulsarlos remita a la dirección del enlace.

Para utilizar una imagen como enlace, colocar la etiqueta <img> dentro de las etiquetas <a></a>.

Advertencia

Al contrario que el ejemplo anterior, un ítem de una lista (<li>) se transforma en enlace englobando solamente el texto en su interior:

<li>Here is <a href="#">a random link</a></li>

Convertir un botón en enlace

Para utilizar un botón como link debe añadirse un fragmento de código JavaScript dentro de la etiqueta button.

Title

El atributo title puede añadirse a los enlaces para añadir información extra. La información se muestra en un pequeño recuadro cuando se sitúa el cursor sobre el enlace.
