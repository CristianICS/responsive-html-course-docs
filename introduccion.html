Introducción

¿Qué es HTML?

  • Lenguaje de marcado o sistema de anotación (HyperText Markup Language)
  • Describe la estructura de una página web
  • Es interpretado por los navegadores
  • La mayoría de anotaciones se basan en etiquetas de inicio y cierre que dotan de sentido al contenido

Origen

Las primeras páginas web, con la creación de la World Wide Web, solían contener documentos estáticos con referencias a otros documentos, compuestas por enlaces con hipertexto. Eran utilizados por el navegador para cargar el documento al que hacían referencia, evitando que el usuario tuviera que buscarlos manualmente.

A medida que las web y sus aplicaciones se hacían más complejas, el grupo W3Consortium actualizó el estándar de HTML para asegurar una visualización correcta de las páginas web en todos los navegadores. La última versión es HTML5.

Lenguaje de marcado

Todo documento HTML se compone de una serie de elementos definidos mediante una etiqueta inicial, el contenido y la etiqueta final.

  • <nombre-etiqueta>Contenido...</nombre-etiqueta>
Advertencia

Algunos elementos HTML no tienen contenido, e.g. etiquetas <br>. Son llamados elementos vacíos, y no requieren de etiqueta de cierre.
