Aplicar estilos

Un objeto HTML puede asociarse con un estilo de tres formas distintas:

  1. Dentro del elemento HTML a modificar
  1. En la cabecera (head) del documento HTML, dentro de la etiqueta style
  1. Escribir los estilos dentro de un archivo .css, llamado en la cabecera de la web mediante la siguiente línea:
    • <link rel="stylesheet" href="estilo/style.css">
Advertencia

Es recomendable acabar todas las declaraciones de estilo con un punto y coma.

Estilos reutilizables

De las tres opciones anteriores, la más adecuada para trabajar es generar una hoja de estilos independiente del archivo html. Por ejemplo, al desarrollar una web puede ser conveniente crear varias páginas adicionales desde las que se podrá acceder a través de la página principal. Con una hoja de estilos se puede aplicar a todas la misma apariencia con tan solo declarar el estilo en el head.
