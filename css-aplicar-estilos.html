Aplicar estilos

Un objeto HTML puede asociarse con un estilo de tres formas distintas:

  1. Dentro del elemento HTML a modificar. Nota: Es una buena práctica acabar las declaraciones con un punto y coma.
  2. En la cabecera (head) del documento HTML, dentro de la etiqueta style
  3. Escribir los estilos dentro de un archivo .css, llamado en la cabecera de la web mediante la siguiente línea:
    4. Advertencia

    Es recomendable acabar todas las declaraciones de estilo con un punto y coma.

    Estilos reutilizables

    De las tres opciones anteriores, la más adecuada es generar una hoja de estilos independiente del archivo html. Por ejemplo, al desarrollar una web puede ser conveniente crear varias páginas adicionales desde las que se podrá acceder a través de la página principal. Con una hoja de estilos se puede aplicar a todas la misma apariencia con tan solo declarar el estilo en el head.

    Un estilo puede asociarse directamente con el nombre de las etiquetas HTML. La siguiente línea de código en un archivo CSS modificará el tamaño de letra de todos los elementos h2:

    h2 {font-size: 20px;}

    Si no se quieren modificar todos los elementos de un mismo tipo, existen tres opciones adicionales para reciclar estilos:

    1. Clases: Una clase consiste en un estilo que puede aplicarse a cada elemento del código HTML. Las clases se definen así dentro del documento CSS:

      Y una vez definida la clase, puede integrarse infinitas veces dentro de las etiquetas HTML.

      Advertencia

      La clase se define en la hoja de estilos comienzando con un punto, pero dentro del atributo class empiza sin el punto.

    2. ID: Modificar estilo mediante su atributo id. Existe un consenso entre los desarrolladores, i.e., utilizar identificadores únicos para cada objeto. Por ello es recomendable no dar a más de un objeto el mismo id. La diferencia entre id y class es que estas últimas son reutilizables y pueden servir para modificar el estilo de varios elementos. Los id tienen mayor especificidad (higher specificity), i.e., si ambos son aplicados a un elemento el estilo que prevalece es el que figura en el id.

    3. CSS Attribute Selectors: mediante la clave [attr=value]. Modifican los elementos que contienen un valor concreto en un atributo. E.g., modificar el estilo de todos los input de tipo radio:

    Selector Ejemplo Explicación
    [attribute] [target] Selecciona todos los elementos con el atributo target
    [attribute=value] [target=_blank] Selecciona todos los elementos con target=_blank
    [attribute~=value] [title=flower] Selecciona todos los elementos con el atributo title que contenga la palabra "flower"
    [attribute|=value] [lang|=en] Selecciona todos los elementos con el atributo lang que empiece por la palabra en. Importante: el valor tiene que ser una palabra entera, o sola o seguida de un guión: "en" o "en-us"
    [attribute^=value] a[href^="https"] Selecciona todos los elementos a cuyo valor en el atributo href comience por "https"
    [attribute$=value] a[href$=".pdf"] Selecciona todos los elementos a cuyo valor en el atributo href termine por ".pdf"
    [attribute*=value] a[href*="w3schools"] Selecciona todos los elementos a cuyo valor en el atributo href contenga la cadena "w3schools"

    La descripción de esta tabla completa de todos los posibles atributos y selectores se puede encontrar en la web de W3School.
